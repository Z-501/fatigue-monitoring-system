# 演示材料说明

建议在本目录或 README 中补充演示视频链接和功能截图。公开展示时不需要暴露源码，只需要让面试官能够直观看到系统链路已经形成。

## 建议演示流程

1. 打开移动端，进入在线疲劳监测页面。
2. 开始一次监测会话，展示疲劳状态或趋势变化。
3. 进入健康管家页面，基于当前状态获取建议。
4. 打开 Web 实验端，展示实验任务、评分或事件记录流程。
5. 展示历史回放或报告页面，说明数据如何沉淀和复盘。

## 建议截图

- 移动端首页 / 监测页。
- 疲劳状态趋势图。
- 健康管家交互页。
- Web 实验任务页。
- 历史回放或报告页。
- 系统服务启动或联调截图。

## 说明

截图和视频应避免出现真实用户信息、真实实验数据、密钥、服务器地址或未脱敏日志。
