# 脱敏接口概览

本文件仅展示接口边界与业务职责，不包含真实域名、密钥、数据库结构或敏感参数。

## 1. 用户与认证

| 接口类别 | 说明 |
| --- | --- |
| 注册 / 登录 | 用户注册、密码校验、Token 生成 |
| 会话校验 | 基于 Token + Redis 的登录态校验 |
| 用户上下文 | 后端统一注入当前用户信息，供在线监测、实验、Agent 和历史模块复用 |

## 2. 在线疲劳监测

| 接口类别 | 说明 |
| --- | --- |
| 设备状态上报 | 上报设备连接、佩戴状态、电量、心率等信息 |
| 基线校准 | 上传校准阶段 EEG 数据，建立个体参考状态 |
| 在线预测 | 按时间窗上传 EEG 数据，返回疲劳分数、风险等级与趋势方向 |
| 当前状态查询 | 查询最近一次疲劳状态，用于前端刷新与 Agent 上下文构造 |

## 3. 离线实验采集

| 接口类别 | 说明 |
| --- | --- |
| 实验 session 创建 | 创建实验会话，生成 Web 与 App 协同所需的 session 信息 |
| 双端 ready | Web 端与 App 端分别上报准备状态 |
| 状态同步 | 后端推进实验状态并同步给两端 |
| 行为事件上传 | Web 端上传任务阶段、trial 结果、主观评分等信息 |
| EEG chunk 上传 | App 端分段上传 EEG 数据 |
| 实验结束归档 | 汇总 EEG、行为事件、评分和元信息，形成可追溯实验记录 |

## 4. Agent 智能干预

| 接口类别 | 说明 |
| --- | --- |
| Agent 对话 | 根据用户输入和当前疲劳上下文生成解释与建议 |
| 主动提醒 | 根据疲劳风险、趋势变化和干预间隔触发提醒 |
| 报告增强 | 基于历史状态和统计结果生成日报/周报解释 |
| Trace / Eval | 记录上下文、Prompt、检索结果、工具调用、模型响应和评估结果 |

## 5. 历史回放与评估

| 接口类别 | 说明 |
| --- | --- |
| 完整历史回放 | 返回当前状态、历史统计和上下文信息 |
| 最近窗口增量 | 返回最近时间窗内的疲劳点，支撑曲线连续刷新 |
| 趋势统计 | 统计 24 小时 / 7 天风险占比、峰值点和突变点 |
| 干预记录 | 查询近期干预事件，供前端展示与 Agent 上下文使用 |
