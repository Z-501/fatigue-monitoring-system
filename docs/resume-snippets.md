# 简历表述参考

## 项目描述

基于 EEG 的全栈疲劳监测与智能健康管家系统，面向办公与实验场景，构建了从移动端监测、Web 实验采集、Java 后端编排、Python 疲劳识别到健康管家 Agent 建议的完整业务闭环。项目以 Java Spring Boot 作为业务中枢，统一处理认证、会话、状态管理、实时推送、历史沉淀和智能服务调用。

## 简历要点

- 负责基于 Java Spring Boot 的业务中枢设计与实现，承接移动端、Web 端、Python 算法服务和 Agent 服务之间的业务编排。
- 设计在线疲劳监测链路，支持监测会话、疲劳状态更新、结果持久化和客户端实时展示。
- 参与实验采集流程设计，支持 APP 与 Web 端协同完成实验任务、事件记录和数据上传。
- 将 EEG 疲劳识别能力以 Python 服务方式接入后端系统，实现算法能力与业务系统解耦。
- 设计健康管家 Agent 的业务入口，将当前疲劳状态、历史记录和用户输入组织为智能建议上下文。
- 整理项目架构图、流程图、接口概览和演示材料，用于项目展示与面试说明。

## 面试概括版

这是一个 AI/算法能力落地到全栈业务系统的项目。它不是单独的模型 Demo，也不是普通 CRUD 后台，而是通过 Java 后端把移动端、Web 实验端、Python 疲劳识别服务、健康管家 Agent 和数据存储连接起来，形成“采集—推理—存储—推送—解释”的完整闭环。
